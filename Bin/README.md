### batch or shell:
- run.bat (run release version in Windows)
- run.sh   (run release version in Linux)
- rund.sh  (run debug version in Linux)
- rund.sh  (run debug version in Linux)
- kill.bat (kill debug & release version in Linux)
- kill.sh  (kill debug & release version in Linux)
